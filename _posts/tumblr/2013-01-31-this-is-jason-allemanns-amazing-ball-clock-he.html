This is Jason Allemann’s amazing ball clock. He has pictures and instructions on how to make many other LEGO sets on his website, though unfortunately no instructions yet on how to make this ball clock.

Allemann also has a maze set on LEGO CUUSOO, a beta LEGO project that allows users to submit and vote for amateur designed sets. Each set that receives 10,000 votes of support is officially reviewed by LEGO for mass production. There is a lot of impressive competition. Alleman’s project could use some support now with only 3,415 votes.
